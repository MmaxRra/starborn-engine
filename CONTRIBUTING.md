## How to Contribute

### 1. Fork the repository
Create **fork** of this repository

### 2. Create branch with your addition
```bash
git checkout -b my-feature
```
### 3. Commit changes to branch
```bash
git commit -m "describe your addition"
```

### 4. Push to branch
```bash
git push origin my-feature
```
You can delete your branch after pull req.
